## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (Icons)

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```

This will build the project and deploy it to the `gh-pages` branch, which GitHub Pages will automatically serve.
